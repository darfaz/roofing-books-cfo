```
Version: 1.0
Effective Date: [DATE]
Last Reviewed: [DATE]
Owner: Office Manager
```

## Purpose

Reduce no-shows and ensure customers are prepared for appointments through systematic confirmation process.

## Scope

All scheduled estimate appointments for residential and commercial roofing.

## Performance Standards

| Metric | Target |
|--------|--------|
| Confirmation Rate | 95% |
| No-Show Rate | <5% |
| Same-Day Reschedule Rate | <10% |

### 2. Day-Before Confirmation

**2.1 Timing**: Between 4:00 PM and 6:00 PM the day before

**2.2 Text Message (Primary)**
```
Hi [First Name]! Just a reminder that [Rep Name] from [Company]
will be at [Address] tomorrow at [Time] for your roof inspection.
Reply YES to confirm or call [Phone] if you need to reschedule.
```

**2.3 If No Response by 7:00 PM**: Phone call attempt

**2.4 Phone Script**
```
"Hi [Name], this is [Your Name] from [Company]. I'm calling to
confirm your appointment tomorrow at [Time] with [Rep Name].
Are we still good to go?"

If confirmed: "Great! [Rep Name] will see you at [Time].
Any questions before then?"

If needs to reschedule: "No problem! When would work better
for you?" [Reschedule and send new confirmation]
```

### 3. Day-Of Confirmation

**3.1 Morning Text** (For afternoon appointments only)
Sent at 9:00 AM for appointments after 12:00 PM:
```
Good morning [First Name]! Looking forward to seeing you today
at [Time]. [Rep Name] will text you when they're on the way. - [Company]
```

**3.2 On-The-Way Notification**
Sales rep sends when departing for appointment:
```
Hi [First Name], this is [Rep Name] from [Company]. I'm on my
way and should arrive at [ETA]. See you soon!
```

**3.3 Arrival Notification**
```
Hi [First Name], I've just arrived at [Address].
I'll knock on the door shortly. - [Rep Name]
```

### 4. No-Show Protocol

**4.1 Definition**: Customer not present 10 minutes after scheduled time

**4.2 Immediate Actions**
1. Call customer's primary phone
2. If no answer, text:
```
Hi [First Name], this is [Rep Name] from [Company]. I'm at
[Address] for our [Time] appointment. Are you on your way?
Please call me at [Phone].
```
3. Wait 5 additional minutes
4. Call secondary phone (if available)
5. Leave voicemail and text on primary

**4.3 Voicemail Script**
```
"Hi [Name], this is [Rep Name] from [Company]. I'm at your
property for our scheduled appointment but wasn't able to
connect with you. I'll wait a few more minutes, but please
call me at [Phone] if you're running late. Otherwise,
I'd be happy to reschedule."
```

**4.4 Door Tag** (if applicable)
Leave company door hanger with handwritten note:
```
Hi [Name] - Sorry I missed you!
I was here for our [Time] appointment.
Please call [Phone] to reschedule.
- [Rep Name], [Company]
```

**4.5 Follow-Up Sequence for No-Shows**
| Timing | Action |
|--------|--------|
| +2 hours | Phone call + text |
| +24 hours | Phone call + email |
| +48 hours | Final attempt - phone + email |
| +72 hours | Close as "No Show - Unresponsive" |

### 5. Reschedule Protocol

**5.1 Customer-Initiated Reschedule**
```
"No problem at all! When would work better for you?
We have availability [Option 1] or [Option 2]."
```

**5.2 Company-Initiated Reschedule**
Minimum 24 hours notice required. Script:
```
"Hi [Name], this is [Your Name] from [Company]. I'm so sorry,
but we need to reschedule your appointment tomorrow due to
[brief reason]. I know your time is valuable, and I want to
make this as easy as possible. Can we move you to [new time]?
As an apology for the inconvenience, [offer small concession
if appropriate]."
```

### 6. Appointment Notes for Sales Rep

Before each appointment, provide rep with:
- [ ] Customer name and contact info
- [ ] Property address with directions
- [ ] Lead source and referrer (if any)
- [ ] Reason for call (repair, replacement, storm, etc.)
- [ ] Previous interactions/notes
- [ ] Insurance information (if applicable)
- [ ] Special instructions or concerns

### 7. Calendar Management

**7.1 Booking Guidelines**
- Allow 90 minutes between residential appointments
- Allow 120 minutes for commercial appointments
- No more than 5 appointments per rep per day
- Block 30 minutes for lunch
- Block drive time for distant appointments

**7.2 Double-Booking Prevention**
- Sync all calendars in real-time
- Check availability before confirming verbally
- Use booking system as source of truth
