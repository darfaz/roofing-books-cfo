## Purpose

Standardize the in-home estimate process to ensure consistent customer experience, accurate pricing, and optimal close rates.

## Scope

All residential and commercial roofing estimate appointments.

## Performance Standards

| Metric | Target |
|--------|--------|
| One-Call Close Rate | 40% |
| Overall Close Rate | 60% |
| Average Appointment Duration | 60-90 min |
| Follow-Up Estimate Delivery | Same day |

## Pre-Appointment Preparation

### Required Equipment
- [ ] Company-branded shirt/uniform
- [ ] Ladder (extension and step)
- [ ] Measuring tools (tape, laser, wheel)
- [ ] Drone (if certified and applicable)
- [ ] Camera or smartphone
- [ ] Tablet/laptop with estimating software
- [ ] Sample boards (shingle colors)
- [ ] Business cards
- [ ] Company brochure
- [ ] Financing application forms
- [ ] Contract/agreement forms
- [ ] PPE (safety glasses, gloves)
- [ ] Door hangers (for no-shows)
- [ ] Roof inspection form
- [ ] Notepad and pen

### Pre-Appointment Review
- [ ] Customer name and pronunciation
- [ ] Property address (confirm in GPS)
- [ ] Lead source and any referrer connection
- [ ] Notes from intake call
- [ ] Previous company history (if any)
- [ ] Satellite measurement (if available)
- [ ] Local permit requirements

## Procedure

### Phase 1: Arrival and Introduction (5-10 minutes)

**1.1 Arrival Protocol**
- Park on street, not in driveway (unless directed)
- Text customer: "I've arrived!"
- Approach front door professionally
- Stand back from door after knocking

**1.2 Introduction Script**
```
[Smile, make eye contact, firm handshake]

"Hi, [Customer Name]? I'm [Your Name] with [Company].
Great to meet you! Thanks for having me out today.
Before we get started, I just want to let you know -
I'm not here to pressure you into anything. My job is
to give you an honest assessment of your roof and answer
any questions you have. Sound good?"
```

**1.3 Build Rapport**
- Comment on something positive about the property
- Ask how long they've lived there
- Ask about the neighborhood
- Listen actively and find common ground

**1.4 Qualify the Opportunity**
Discovery questions (conversational, not interrogation):
- "What made you decide to get your roof looked at?"
- "How long have you been thinking about this?"
- "Are you comparing a few companies?"
- "When are you hoping to have this done?"
- "Is there anyone else involved in this decision?"
- "Have you thought about how you'd like to pay for this?"

### Phase 2: Property Assessment (20-30 minutes)

**2.1 Ground-Level Inspection**
Walk perimeter and note:
- [ ] Roof visibility from ground
- [ ] Gutter condition
- [ ] Downspout placement
- [ ] Siding condition near roofline
- [ ] Landscape obstacles
- [ ] Access points for crew

**2.2 Roof Inspection**
Climb roof (if safe) and inspect:
- [ ] Overall condition rating (1-10)
- [ ] Shingle type and condition
- [ ] Approximate age
- [ ] Layer count
- [ ] Damaged/missing shingles
- [ ] Nail pops
- [ ] Granule loss
- [ ] Curling or buckling
- [ ] Flashing condition
- [ ] Valley condition
- [ ] Ridge and hip condition
- [ ] Pipe boot condition
- [ ] Chimney condition
- [ ] Skylight condition
- [ ] Ventilation assessment
- [ ] Visible decking issues
- [ ] Storm damage indicators

**2.3 Measurements**
- Roof planes with dimensions
- Ridge, hip, and valley lengths
- Eave and rake lengths
- Pitch measurement
- Special features (dormers, etc.)

**2.4 Interior Inspection (if accessible)**
Attic inspection when possible:
- [ ] Daylight visible through roof
- [ ] Water stains or damage
- [ ] Ventilation assessment
- [ ] Insulation condition
- [ ] Structural concerns

**2.5 Photo Documentation**
Required photos:
- [ ] Front of home (full view)
- [ ] Overall roof shots (each plane)
- [ ] Problem areas (close-up)
- [ ] Flashings and penetrations
- [ ] Gutters and edges
- [ ] Any damage found
- [ ] Interior issues (if any)

### Phase 3: Estimate Building (10-15 minutes)

**3.1 Return to Customer**
```
"Thanks for your patience while I completed the inspection.
I found some things I want to show you. Do you have a few
minutes to look at some photos with me?"
```

**3.2 Show Findings**
- Walk through photos on tablet
- Explain issues in simple terms
- Avoid scare tactics - be factual
- Answer questions thoroughly

**3.3 Build Estimate**
Using company pricing/estimating software:
- Calculate material quantities
- Apply labor rates
- Add permits and miscellaneous
- Calculate options (Good/Better/Best)
- Apply current promotions

### Phase 4: Presentation (20-30 minutes)

**4.1 Set the Stage**
```
"Based on what I found, let me show you a few options.
I always like to give homeowners choices so you can
pick what makes the most sense for your situation
and budget."
```

**4.2 Present Options (Good/Better/Best)**

Start with the BETTER option (most recommended):
```
"Most of our customers choose this option because...
[explain value and why it's the best fit]"
```

Then present GOOD:
```
"If budget is the primary concern, this option gives you...
[explain what they get and what they give up]"
```

Then present BEST:
```
"For maximum protection and longest life, this option
includes... [explain premium benefits]"
```

**4.3 Explain What's Included**
Walk through scope of work:
- Complete tear-off
- Deck inspection and repairs
- Underlayment installation
- Shingle installation
- Flashing replacement
- Cleanup and nail sweep
- Warranties
- Timeline

**4.4 Address Investment**
```
"Your investment for [option] would be [amount].
That includes everything we discussed plus our
[X]-year workmanship warranty."
```

**4.5 Financing Introduction**
```
"By the way, we do offer financing if that would
make this easier to fit in your budget. With
approved credit, this could be as low as [amount]
per month. Would you like me to show you those options?"
```

### Phase 5: Closing (10-15 minutes)

**5.1 Trial Close**
```
"Based on what we discussed, which option makes
the most sense for you?"
```

**5.2 Handle Objections**
See SALES-006: Objection Handling Guide

Common objections:
- "I need to think about it" → "What specifically do you need to think about?"
- "I need to talk to my spouse" → "I understand. Is there anything I can explain that would help that conversation?"
- "I'm getting other quotes" → "I respect that. What would help you make the best decision?"
- "It's too expensive" → "I hear you. Let me show you the financing options..."

**5.3 If Ready to Move Forward**
```
"Great! Let me walk you through the next steps.
[Pull out agreement and review terms]"
```

Complete paperwork:
- [ ] Sign contract/agreement
- [ ] Collect deposit
- [ ] Schedule tentative start date
- [ ] Explain what happens next
- [ ] Leave contact information

**5.4 If Not Ready Today**
```
"I completely understand wanting to take some time.
I'll email you the estimate today so you have all
the details. When would be a good time for me to
follow up with you?"
```

Schedule specific follow-up:
- Get commitment to date/time
- Note on calendar
- Enter in CRM

### Phase 6: Wrap-Up

**6.1 Departure**
```
"Thank you so much for your time today. It was
great meeting you. Do you have any other questions
before I head out?"
```

Leave behind:
- Business card
- Company brochure
- Printed estimate (if closed)
- Shingle color sample (if requested)

**6.2 Post-Appointment Tasks (within 1 hour)**
- [ ] Send thank-you text
- [ ] Email estimate with cover letter
- [ ] Update CRM with full notes
- [ ] Upload photos
- [ ] Schedule follow-up tasks
- [ ] Report results to sales manager
