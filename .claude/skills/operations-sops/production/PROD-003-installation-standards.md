## Purpose

Define installation standards and procedures to ensure consistent, high-quality roofing work that meets manufacturer specifications, building codes, and customer expectations.

## Scope

All residential shingle roof installations including tear-off and replacement.

## Reference Standards

- IRC (International Residential Code) - Chapter 9
- Manufacturer installation instructions (always follow)
- Local building codes (may be more restrictive)
- OSHA safety requirements

## Daily Job Start Procedure

### 1. Arrival Protocol

**1.1 Crew Arrival (All arrive together)**
- Park vehicles to not block customer
- No music audible outside vehicle
- Professional appearance required

**1.2 Site Assessment**
Before starting any work:
- [ ] Walk perimeter of property
- [ ] Identify hazards
- [ ] Note obstacles and sensitive areas
- [ ] Photograph any pre-existing damage
- [ ] Check for overhead power lines

**1.3 Customer Contact**
If customer is home:
```
"Good morning! I'm [Name], crew lead from [Company].
We're here to start your roof. Before we begin, is
there anything specific I should know about? Any
areas you'd like us to be careful around?"
```

**1.4 Daily Safety Briefing**
See SAFETY-001 - Complete before work begins

### 2. Tear-Off Procedure

**2.1 Property Protection**
Before tear-off:
- [ ] Place plywood/protection over A/C unit
- [ ] Cover plants and landscaping with tarps
- [ ] Move patio furniture away from house
- [ ] Set up debris catch at eaves (if applicable)
- [ ] Clear dumpster access path

**2.2 Tear-Off Sequence**
1. Start at ridge, work down
2. Remove ridge cap and hip shingles
3. Remove field shingles in sections
4. Remove flashing around penetrations
5. Remove drip edge
6. Remove old underlayment
7. Remove protruding nails
8. Expose and inspect deck

**2.3 Debris Management**
- Keep roof clear of debris (trip hazard)
- Load debris directly into dumpster when possible
- No debris on customer's landscaping
- Keep walkways clear at all times
- Clean up before leaving each day

### 3. Deck Inspection and Repair

**3.1 Deck Inspection Checklist**
Walk entire deck and inspect for:
- [ ] Soft or spongy areas (water damage)
- [ ] Delaminated or warped sheathing
- [ ] Holes or missing sections
- [ ] Improper nailing patterns
- [ ] Gaps between sheets
- [ ] Structural issues (sagging, broken rafters)

**3.2 Deck Repair Standards**
Replace sheathing when:
- More than 1/4" of deflection under load
- Signs of rot or water damage
- Delamination present
- Structural integrity compromised

Repair procedure:
1. Cut out damaged section
2. Ensure cuts are centered on rafters
3. Install new sheathing of equal or greater thickness
4. Nail per code (8d nails, 6" OC edges, 12" OC field)
5. Document repairs with photos

**3.3 Change Order Requirement**
Any deck repair exceeding allowance must:
- Stop work
- Document with photos
- Call customer/sales rep
- Get approval before proceeding
- Document change order

### 4. Underlayment Installation

**4.1 Ice & Water Shield**
Install ice & water shield at:
- [ ] Eaves: Minimum 24" beyond interior wall line
- [ ] Valleys: Full length, minimum 36" wide
- [ ] Around penetrations: As required
- [ ] Low-slope areas: Per code requirements
- [ ] Skylights: Full perimeter

Installation standards:
- Clean, dry deck surface
- Remove release paper in sections
- Roll for full adhesion
- Overlap seams minimum 6"
- Extend 1/4" over drip edge

**4.2 Synthetic Underlayment**
Install over remaining roof area:
- Start at eaves, work up
- Overlap horizontal seams per manufacturer (typically 4")
- Overlap vertical seams per manufacturer (typically 6")
- Secure with cap nails or staples per spec
- Keep taut, no wrinkles or bubbles
- Extend 2" over hips and ridges

### 5. Drip Edge Installation

**5.1 Drip Edge Standards**
- Minimum 2" face
- Extend 1/4" beyond fascia
- Install over underlayment at rakes
- Install under underlayment at eaves
- Overlap joints minimum 2"
- Nail every 12" (staggered)

**5.2 Installation Sequence**
1. Install eave drip edge first (under underlayment)
2. Install underlayment
3. Install rake drip edge (over underlayment)

### 6. Starter Strip Installation

**6.1 Starter Strip Placement**
- Install along all eaves and rakes
- Adhesive strip positioned toward roof edge
- Overhang eave drip edge 1/4" to 3/4"
- Cut tabs to stagger from first course

**6.2 Fastening**
- 4-6 nails per starter strip
- Position nails in adhesive line area
- Do not nail in single-coverage areas

### 7. Shingle Installation

**7.1 General Requirements**
- Follow manufacturer specifications exactly
- Minimum 4 nails per shingle (6 for high wind)
- Proper nail placement in manufacturer's nail line
- Maintain specified exposure
- Stagger cutouts minimum 4" (architectural)

**7.2 First Course**
- First full shingle over starter strip
- Offset cutouts from starter
- Align with chalk line
- Maintain consistent overhang

**7.3 Pattern and Alignment**
- Snap chalk lines every 2-3 courses
- Maintain consistent exposure (typically 5-5/8")
- Random stagger for architectural shingles
- Minimum 4" cutout offset (3-tab)

**7.4 Nail Placement**
Proper nailing is critical:
- In manufacturer's nail line
- Straight, flush, not overdriven
- Not in cutouts or sealant strip
- High-wind areas: 6 nails

**7.5 Working Around Penetrations**
- Cut shingles to fit around obstacles
- Do not stretch or force fit
- Maintain proper nail placement
- Allow for thermal movement

### 8. Flashing Installation

**8.1 Valley Flashing**
Open valleys:
- Minimum 24" wide metal
- No nails within 6" of centerline
- Shingles trimmed at angle
- Sealant on back edge

Closed valleys:
- Weave or closed cut method
- No nails within 6" of centerline
- Ice & water shield beneath

**8.2 Pipe Boot Installation**
1. Install boot over existing vent
2. Position base under shingles above
3. Position base over shingles below
4. Nail at corners
5. Apply sealant at edges
6. Install shingles around boot

**8.3 Step Flashing**
- 4" x 5" minimum (5" x 7" preferred)
- Install with each shingle course
- Do not nail through surface
- Extend 2" beyond shingle
- 2" minimum up wall

**8.4 Counter Flashing**
- Covers top of step flashing
- Embedded or surface-mounted
- Properly sealed at wall

### 9. Ridge and Hip Installation

**9.1 Ridge Vent**
If installing ridge vent:
1. Cut ridge opening per manufacturer
2. Install baffles if required
3. Install vent per specifications
4. Install ridge cap over vent

**9.2 Ridge Cap Shingles**
- Center on ridge/hip
- Nail in manufacturer's nail zone
- Overlap per manufacturer spec
- Direction: Away from prevailing wind

### 10. Ventilation

**10.1 Ventilation Balance**
Verify proper intake and exhaust:
- 1 sq ft NFA per 150 sq ft attic (without vapor barrier)
- 1 sq ft NFA per 300 sq ft attic (with vapor barrier)
- 50/50 split intake/exhaust ideal

**10.2 Vent Installation**
- Do not block existing vents
- Do not mix exhaust vent types
- Ensure intake vents clear

### 11. Quality Checkpoints

**Throughout Installation:**
- [ ] Nail placement correct
- [ ] Exposure consistent
- [ ] Alignment straight
- [ ] Flashings properly sealed
- [ ] No visible defects

**Completion Checklist:**
See QC-001: Final Inspection Checklist

### 12. Daily Cleanup

**Every Day Before Leaving:**
- [ ] Magnetic nail sweep entire property
- [ ] Remove debris from landscaping
- [ ] Clear walkways
- [ ] Cover any exposed areas if not complete
- [ ] Remove loose materials from roof
- [ ] Organize job site
- [ ] Lock up materials

- Change orders

**Photo Documentation:**
- Before (pre-existing conditions)
- Deck repairs
- Installation progress
- Completed work
