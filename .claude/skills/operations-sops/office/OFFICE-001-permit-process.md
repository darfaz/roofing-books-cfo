## Purpose

Ensure proper permitting for all roofing work requiring permits, maintaining compliance with local building codes and protecting customers.

## Scope

All roofing work in jurisdictions requiring permits, typically:
- Complete roof replacements
- Structural repairs
- Commercial roofing
- Work affecting fire ratings

## Procedure

### 1. Permit Requirement Check

**1.1 Determine If Permit Required**
At contract signing, check:
- [ ] Project location (jurisdiction)
- [ ] Type of work (repair vs. replacement)
- [ ] Scope of work (squares involved)
- [ ] Any structural changes

**1.2 Permit Decision**
Document in job file:
- Permit required: Yes / No
- Jurisdiction: _______________
- Permit type: _______________
- Estimated fee: $ ___________

### 2. Permit Application

**2.1 Required Information**
Gather before applying:
- [ ] Property address
- [ ] Property owner name
- [ ] Owner mailing address (if different)
- [ ] Contractor license number
- [ ] Contractor insurance info
- [ ] Scope of work description
- [ ] Materials to be used
- [ ] Roof measurements
- [ ] Number of layers to be removed
- [ ] Value of work

**2.2 Application Methods**

**Online Application (Preferred)**
1. Log into jurisdiction portal
2. Create new permit application
3. Enter required information
4. Upload any required documents
5. Pay fee online
6. Save/print confirmation

**In-Person Application**
1. Print application form
2. Complete all fields
3. Attach required documents
4. Deliver to permit office
5. Pay fee
6. Obtain permit or receipt

**2.3 Standard Application Description**
```
Scope of Work:
Remove [X] layer(s) of existing roofing material.
Inspect and repair roof deck as needed.
Install new [underlayment type] underlayment.
Install new [shingle brand/type] asphalt shingles.
Replace all flashings and pipe boots.
Install new drip edge on all eaves and rakes.
Install ridge vent as needed for proper ventilation.
Total area: approximately [X] squares.
```

### 3. Permit Tracking

**3.1 Permit Log**
Maintain log of all permits:

| Job | Address | Jurisdiction | Applied | Issued | # | Expires | Final |
|-----|---------|--------------|---------|--------|---|---------|-------|
| | | | | | | | |

**3.2 Application Status Check**
- Check status within 48 hours of application
- Follow up if not issued within expected timeframe
- Document all communications

### 4. Permit Display and Compliance

**4.1 Permit Posting**
When permit is issued:
- [ ] Print permit if not hard copy
- [ ] Include in job packet
- [ ] Instruct crew to post on-site
- [ ] Permit must be visible during inspection

**4.2 Crew Instructions**
```
PERMIT INSTRUCTIONS FOR CREW:

□ Post permit in visible location before starting work
□ Keep permit dry and protected
□ Do not remove until final inspection
□ Call office if inspector arrives
□ DO NOT COVER WORK before inspection if required
```

### 5. Inspection Scheduling

**5.1 Inspection Types**

| Inspection | When Required | Timing |
|------------|---------------|--------|
| Pre-work | Some jurisdictions | Before starting |
| Deck/Sheathing | Structural repairs | Before covering |
| Final | All permits | After completion |

**5.2 Scheduling Process**
1. Complete required work phase
2. Call inspection line: [Phone]
   Or schedule online: [URL]
3. Request inspection type
4. Note confirmation number
5. Inform customer of inspection date

**5.3 Inspection Request Script**
```
"I need to schedule a [type] inspection.
Permit number: [Number]
Address: [Address]
Contractor: [Company Name]
Contact: [Your Name/Phone]
Work completed: [Date]"
```

### 6. Inspection Day

**6.1 Pre-Inspection Preparation**
- [ ] Work area accessible
- [ ] Permit posted and visible
- [ ] Roof accessible (ladder in place)
- [ ] No safety hazards
- [ ] Customer aware of inspection

**6.2 Inspection Results**

**If Passed:**
- Obtain signed inspection card
- File in job folder
- Proceed with project closeout

**If Failed/Corrections Required:**
1. Document required corrections
2. Understand what needs to be fixed
3. Schedule crew for corrections
4. Reschedule inspection
5. Notify customer if delays

### 7. Permit Closeout

**7.1 Final Documentation**
Ensure job file contains:
- [ ] Copy of permit
- [ ] Inspection card (if provided)
- [ ] Passed inspection documentation
- [ ] Any variance or special approvals

**7.2 Permit Close**
Some jurisdictions require:
- Final documentation submission
- Online permit closure
- Certificate of completion upload

### 8. Special Circumstances

**8.1 Permit Not Yet Issued**
DO NOT start work without permit when required.
If delayed:
- Contact permit office
- Explore expedite options
- Consider re-scheduling job

**8.2 Owner-Pull Permits**
Some jurisdictions allow homeowner permits:
- Provide owner with information
- Owner goes to permit office
- Owner is permit holder (responsibility)
- Document in contract

**8.3 Work Without Permit (Discovered)**
If work started without required permit:
1. Stop work immediately
2. Apply for permit same day
3. Request inspection of work in place
4. Be prepared for additional fees
5. Document incident
6. Identify process failure

### 9. Common Issues and Solutions

| Issue | Solution |
|-------|----------|
| Permit denied | Review requirements, resubmit correctly |
| Long wait times | Apply earlier in process |
| Inspection failed | Fix issues, reschedule |
| Inspector unavailable | Request specific date |
| Permit expired | Apply for extension or renewal |
