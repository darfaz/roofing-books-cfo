# Matador Driver Model (Roofing)

## Drivers
1. Management Independence
2. Financial Records Quality
3. Recurring Revenue
4. Operational Systems
5. Customer Base Diversity
6. Market Outlook

## Scoring
- Each driver scored 0–100
- Scores roll up into tier:
  - Below Average
  - Average
  - Above Average

## Tier → Multiple Mapping
- Below Avg: ~3× EBITDA
- Average: ~4.5–5× EBITDA
- Above Avg: ~7×+ EBITDA

## Evidence & Audit
- Each score requires evidence or telemetry
- Tier changes that materially impact valuation require owner acknowledgement
